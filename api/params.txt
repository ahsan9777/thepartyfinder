- Get Event Post Data:
URL: api/index.php?action=get_post

Parameter ( Optional )
event_index ( Set the value in the parameter to get the specific data against the events like Nightclub otherwise show all events )
post_id ( Set the value in the parameter to get the specific data against the post_id otherwise, show all events )

------------------------------------------------------------
